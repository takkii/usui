## 雨水 (usui)

> 作者情報

> zip   圧縮

> unzip  解凍

**詳しくは、[タグ屋さんの平台](http://rubyist.hatenablog.com/entry/2016/09/28/203450)を参照してください**

## コンパイル

> git clone https://github.com/takkii/usui.git

> bundle install

> bundle exec warble jar .

**usui.jarが生成されます**
